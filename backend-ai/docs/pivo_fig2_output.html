C · PWA supply simulator (client-side, no API call)
Owner drags a slider to set Y (units to prepare). Recalculates in real time from the JSON payload. Uses qty_mid as demand baseline. No extra API call needed.
Exact prep (Y = demand)
profit = (price - cost) x Y
No waste, no stockout
Underprep (Y < demand)
profit = (price - cost) x Y opp_loss = (demand - Y) x (price - cost)
Sold out — missed revenue shown
Overprep (Y > demand)
profit = (price - cost) x demand waste_loss = (Y - demand) x unit_cost
Waste cost shown in IDR
Inputs from JSON payload: qty_mid, unit_price, unit_cost. Pure client-side arithmetic in React PWA.
